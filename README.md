
## Instale o BOT:

```bash
> termux-setup-storage
> pkg install git
> pkg install ffmpeg
> pkg install wget
> pkg install nodejs
> git clone (Link do Git) 
> cd Caussx
> bash install.sh
```

### Para rodar
```bash install.sh
> npm start
```
### Para Usar PM2
```bash
> npm i -g pm2 (para instalar)

> Para executar = pm2 start node index.js
> depois = pm2 monit
```
